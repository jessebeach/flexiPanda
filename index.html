flexiPanda by jessebeach

A responsive, event-driven dropdown menu jQuery plugin

Authors

Jesse Beach (splendidnoise@gmail.com)

Demonstrations

  • Simple horizontal dropdown menu with hover example.
  • Simple horizontal dropdown menu with hover example for RTL languages.
  • Simple horizontal dropdown menu with hover example with one primary level.
  • Debug information turned on.

$ git clone git://github.com/jessebeach/flexiPanda
